welcome to my world

第一次写博客，不像网上教程般所说搭建博客只需几小时，我几乎断断续续折腾了半年，如今终于有了雏形，个中曲折是难以言说的，只得慢慢记下，以免日后踩坑，也希望大家可以从中吸取些许教训。

  • Git的安装
  • node.js的安装
  • hexo的安装配置
  • Git与GitHub连接
  • Git bash命令行部署博客

参照此教程较为不错：github+hexo搭建博客教程

What’s more,notice：

坑1：仓库的名字

仓库的名字必须是：GitHub用户名.github.io，否则报错

坑2：Git不是GitHub

Git是指Git bash而不是GitHub desktop或者Github

坑3：Hexo命令

Hexo命令在Git bash里打或者cmd里面打都可以，但：Git要运行在本地Hexo的根目录下

坑4：_config.yml配置文件语法严格

  • “：”后面一定要加上一个空格

  • 编码采用UTF-8，否则中文会乱码

坑5：hexo部署命令后，网站没动静

  • 博客第一次创建，即使deploy成功了，但仍然不会立刻可以访问，还需等15分钟以上
  • 再试着hexo deploy一次
  • 到yourname.github.io项目的github主页，点击settings，拉到github pages，有一个launch automatic page generator，点击后，再点击continue to layouts，最后点击publish page，马上就可以看到yourname.github.io可以访问了，但样式是刚刚默认的，和本地不一样

Last but not least

  • 更换主题themes
  • 写博客，Markdown软件，采用在线的Madoko配合editplus即可

and so on！
