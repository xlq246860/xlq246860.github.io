NetBeans and MySQL

发表于 2018-09-03 | 分类于 Java ee

netbeans的安装

一路默认即可，但不要一同安装glassfish服务器，将glassfish服务器单独放在C盘根目录下，（PS：netbeans最好不要安装在C盘，我的安装位置：D:\Downloads\netbeans）

MySQL的安装

可谓是峰回路转，九九八十一难呀！

因为之前安装过MySQL的command line版，所以想安装MySQL workbench版，却发现二者或还是不兼容吧。没办法只得卸载——>重装，再卸载——>再重装……无穷尽也。

MySQL之卸载：

  • 控制面板卸载
  • 文件夹卸载

c:\program files\mysql

c:\programdata\mysql(programdata文件夹为隐藏文件夹)

  • 注册表删除,cmd运行“regedit”，打开注册表

HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Services\Eventlog\Application\MySQL文件夹
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Eventlog\Application\MySQL的文件夹

这便算卸载干净了，重启便应该可以了！

MySQL之重装

双击.msi文件一路默认便是最简单不过的了，不过这回导致文件过大，但因需要workbench的可视化界面，便只好为难地默认安装，若是不需要可视化，可只安装mysql server即可。
